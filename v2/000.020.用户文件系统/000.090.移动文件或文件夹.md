## 删除文件或文件夹

* 路径: ```/v2/files/move```
* 方法: ```POST```
* 登录: 需要

#### 需要提交的字段


文件路径的的数组

#### 注意事项

* source 与 destination 不可缺

#### 示范

* 发送

```POST``` ```/v2/files/move```

```json
{
    "source": [
        {
            "path": "/b"
        }
    ],
    "destination": {
        "path": "/f"
    }
}

```

或

```json
{
    "source": [
        {
            "identity": "dedededwedwefge"
        }
    ],
    "destination": {
        "identity": "dedededwedwefge"
    }
}

```

* 接收：

```json
{
    "status": 0,
    "result":  2,
    "code": "OK",
    "success": true
}
```

* result 字段用以指示成功移动的文件数量。
