### 列出回收站内容（分页方式）

## 以分页方式列出文件夹

* 路径: ```/v2/trash/list```
* 方法: ```POST```
* 登录: 需要

#### 需要提交的字段

| 字段          	| 类型    	| 默认值 	| 必填 	| 说明               	| 备注                         	|
|---------------	|---------	|--------	|------	|--------------------	|------------------------------	|
| start          	| long  	|        	| 是   	| 从第几行开始             	|   默认为0   	|
| listSize         	| long  	|        	| 是   	| 单页显示的数量          	|   默认为20   	|



#### 示范

1. 获取回收站信息

* 发送

```POST``` ```/v2/trash/page```

```json
{
	"start":0,
	"listSize": 1
}
```

* 接收：

```json
{
    "status": 0,
    "result": {
        "list": [
            {
                "identity": "5c684513531086e221628327",
                "name": "IPTD-999",
                "source": "/IPTD-999",
                "size": 0,
                "deleteTime": 1556491822891
            }
        ]
    },
    "code": "OK",
    "success": true
}
```
