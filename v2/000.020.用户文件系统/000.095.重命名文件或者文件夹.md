## 重命名文件或者文件夹

* 路径: ```/v2/user/rename```
* 方法: ```POST```
* 登录: 需要

#### 需要提交的字段

"identity" / "path"
"name"

文件路径的的数组

#### 注意事项

* 接受文件路径或者ID

#### 示范

* 发送

```POST``` ```/v2/user/rename```

```json
{
    
    "path": "/b",
    "name": "papaya"
}

```

或

```json
{
    "identity": "dedededwedwefge",
    "name": "papaya"
}

```

* 接收：

```json
{
    "status": 0,
    "result":  true,
    "code": "OK",
    "success": true
}
```

* result 字段用以指示是否重命名成功。
* 重命名文件是异步操作，因此可能会出现刚刚重命名完成的文件夹里面的内容无法显示，稍等即可恢复正常。
