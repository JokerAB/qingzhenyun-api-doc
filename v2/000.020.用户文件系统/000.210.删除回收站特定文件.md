## 删除文件或文件夹

* 路径: ```/v2/trash/delete```
* 方法: ```POST```
* 登录: 需要

#### 需要提交的字段


需要删除的ID

#### 注意事项

* 接受任务ID数组

#### 示范

* 发送

```POST``` ```/v2/trash/delete```

```json
["27594a4f3dbd1b66a787c63190287eb7","dda0b4e445c73dfd32832d8c683e5fa6"]

```

或

```json
{
    "identities": [
        "27594a4f3dbd1b66a787c63190287eb7","dda0b4e445c73dfd32832d8c683e5fa6"
    ]
}

```

* 接收：

```json
{
    "status": 0,
    "result": 2,
    "code": "OK",
    "success": true
}
```
