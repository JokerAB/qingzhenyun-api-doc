## 添加离线任务

* 与 v1相比，一次请求即开启离线任务，更加简洁。

### 1、预解析下载任务

* 路径: ```/v2/offline/parseUrl```
* 方法: ```POST```
* 登录: 需要

#### 需要提交的字段

| 字段          	| 类型    	| 默认值 	| 必填 	| 说明               	| 备注                         	|
|---------------	|---------	|--------	|------	|--------------------	|------------------------------	|
| url   	| string  	|      	| 默认 	| 路径，一行一个               	| 下载链接                  	|

* "\n" 用以区分每个链接

```POST``` ```/v2/offline/parseUrl```

### 2、发起下载任务

* 路径: ```/v2/offline/add```
* 方法: ```POST```
* 登录: 需要

#### 需要提交的字段

| 字段          	| 类型    	| 默认值 	| 必填 	| 说明               	| 备注                         	|
|---------------	|---------	|--------	|------	|--------------------	|------------------------------	|
| task   	| 对象数组  	|      	| 默认 	| =               	| 上次传回的对象                  	|


* task说明
[   数组外侧
    {
        identity : string, 上次接口返回
        iginreFiles: string[], 选择需要下载的文件，如果传空则都下载
    }

]
