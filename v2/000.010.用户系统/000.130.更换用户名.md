## 更改用户名

* 登录时， 可以更改用户名。

* 路径: ```/v2/user/changeName```
* 方法: ```POST```
* 登录: 需要

#### 需要提交的字段


| 字段          	| 类型    	| 默认值 	| 必填 	| 说明               	| 备注                         	|
|--------------	|---------	|--------	|------	|--------------------	|------------------------------	|
| name             | string  	|        	| 是   	| 新用户名            | 用户名不可和已有用户重复    	|

#### 返回

布尔值

POST
```json
{
	"name": "@@@F@大"
}
```

接收

```json
{
    "status": 0,
    "result":  true,
    "code": "OK",
    "success": true
}
```
